
# 🎵 Spotify Music Streaming Analytics

## 📌 Project Overview

The **Spotify Music Streaming Analytics** project focuses on analyzing music streaming data to uncover listener preferences, identify trending genres, evaluate artist performance, and generate business insights. Using **SQL**, **Python**, and **Power BI**, the project transforms raw Spotify song data into meaningful analytics that can support business decision-making and improve music recommendation systems.

## 🎯 Business Problem

Music streaming platforms generate large volumes of user listening data. Understanding listener behavior and music trends helps improve user engagement, personalize recommendations, and support strategic marketing decisions.

## 🎯 Project Objectives

* Analyze music streaming behavior.
* Identify trending music genres.
* Evaluate artist performance.
* Understand listener preferences.
* Improve recommendation systems using audio features.
* Generate actionable business insights through interactive dashboards.

## 🛠️ Tools & Technologies

* **MySQL** – Data querying and analysis
* **Python** – Data analysis and visualization
* **Power BI** – Interactive dashboards
* **Microsoft Excel** – Initial data review
* **Kaggle** – Dataset source

## 📂 Dataset

**Dataset Name:** Spotify Songs Dataset

The dataset contains song-level information including:

* Track Name
* Artist Name
* Playlist Genre
* Track Popularity
* Danceability
* Energy
* Acousticness
* Valence
* Tempo
* Other audio features
